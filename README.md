# 🏠 Projekt Sauerbruch 3 - Hausplanung & Renovierung

Dieses Repository ist das zentrale Tool zur Planung, Dokumentation und Kostenüberwachung der Renovierungsmaßnahmen in der Sauerbruchstraße 3.

🚀 **Live-Planer:** [https://del4gent.github.io/sauerbruch/](https://del4gent.github.io/sauerbruch/)

## 🛠 Projekt-Fokus
Die aktuelle Phase konzentriert sich auf die Renovierung der folgenden Bereiche:
- 🚿 **Bad** (In Arbeit - Abriss erfolgt)
- 🚽 **Gästebad** (In Planung)
- 🧣 **Flur** (In Planung)
- 🛋 **Wohnzimmer** (In Planung)
- 📦 **Kellerflur** (In Planung)

## 📏 Berechnungs-Prinzip (Wichtig!)
Gemäß unserer Qualitätsrichtlinien gilt für alle Daten in diesem Projekt:
**Jede berechnete Zahl muss zwingend eine Herleitung enthalten.**
- **Flächen:** Z.B. `15 m² (Derivation: 3.75m x 4.00m)`
- **Kosten:** Z.B. `1.200 € (Material: 800€ + Arbeit: 400€)`

## 💻 Tech Stack
- **Frontend:** Angular 21 (Nx Monorepo)
- **Styling:** Modernes Dark-Mode Interface (Vanilla CSS / Tailwind-inspiriert)
- **Deployment:** Automatisiert via GitHub Actions auf GitHub Pages
